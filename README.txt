# GOBOG Gold Website (Static + Editable Config)

Edit `config.json` saja untuk:
- TOKEN_ADDRESS
- PRESALE_ADDRESS
- USDT_ADDRESS
- CHAIN_ID (97 testnet / 56 mainnet)
- EXPLORER_BASE

Deploy: GitHub Pages -> main /(root)

Catatan:
- Parameter presale di blockchain tetap immutable.
- Website bisa kamu update kapan saja (alamat kontrak, twitter, jaringan) lewat config.json.
